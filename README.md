### Project Overview
A 5-page website developed for the smile Foundation to facilities surgical enquiries and donor engagement.

### Key features
* Semantic HTML5 structure (header, nav, main, footer)
*   Functionnal 5-page navigation
* Patient Enquiry Form with validation.
* Responsive design for cross-browser compatibility.

### Sitemap
1. Home (index.html)
2. About Us (about.html)
3. Services (services.html)
4. Enquiry (enquiry.html)
5. Contact (contact.html)

### Website Goals and Objectives
* To raise awareness for the Smile Foundation's surgical initiatives.
* To provide a streamlined digital enquiry form for parents and guardians.
* To offer clear contact information for multiple regional offices.
